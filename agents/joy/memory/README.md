# Joy Memory Log

This directory contains Joy's daily memory files.

## File Naming

- `YYYY-MM-DD.md` — Daily activity log
- `email-actions.log` — Email processing audit trail (in ../logs/)

## What Gets Logged

- Email check results
- Emails processed (trusted senders)
- Emails queued for review
- Errors or issues
- Cron job status

## Format

```markdown
# 2026-03-31 — Joy Activity Log

## Email Checks
- [10:00] Checked inbox — 3 new emails, 2 processed, 1 queued
- [10:10] Checked inbox — 0 new emails
- [10:20] Checked inbox — 1 new email, 1 processed

## Summary
- Total emails processed: 3
- Total queued for review: 1
- No issues encountered
```
